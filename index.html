About the Ubuntu Font Family

<
title

The Ubuntu font family's development is funded and used by Canonical, the company behind the popular Linux operating system of the same name. Currently, the technical work of the font family is being handled by Dalton Maag, a well-known UK-based typeface developer.

This font family is designed to represent the attitudes that Ubuntu's brand embodies: precision, reliability, and freedom.

The Ubuntu font family is currently comprised of three style families: Ubuntu Standard, Ubuntu Condensed, and Ubuntu Mono. Ubuntu Standard is designed for body type, while Ubuntu Condensed and Ubuntu Mono are used for display sections. Together, these style families provide 13 different style options to choose from as seen in the sections below.
